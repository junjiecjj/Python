# fractional_AF_generator
Simulation codes to generate the fractionally-interpolated ambiguity function cuts of general discrete-time waveforms, as presented in the article: "Normalized Ambiguity Function Characteristics of OFDM, OTFS, AFDM, and CP-AFDM for ISAC" submitted to the IEEE ICC 2026.
